# 麻雀役道場

麻雀の役を学べるWebアプリです。

**https://mizunokura.github.io/mahjong-yaku-trainer/**

## モード

### 道場モード

牌を引いて捨てる実践形式で、手牌から狙える役とその距離をリアルタイムに確認できます。

- 打牌シミュレーション: 牌をタップすると、切った場合の役への影響をプレビュー
- ポン・チーのプレビュー: 他家の捨て牌に対する鳴き判断を比較
- 向聴数・受入表示: テンパイまでの距離と待ち牌/受入牌を表示
- 点数計算: 和了時に翻数・符数・点数を自動計算
- 狙う役の固定: 特定の役をピンして上部にハイライト

### クイズモード

14枚の完成手牌が表示され、成立している役を全て選ぶクイズです。

- レベルに応じた出題（1翻〜役満）
- 不正解時は各役の構成牌を構造的に解説（順子・雀頭・両面待ちなど）
- 正解率のトラッキング
